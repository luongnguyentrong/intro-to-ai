### Intro to AI course

#### Running 
_Easily set up a local development environment !_
- Install the dependencies: `pip install -r requirements.txt`
- Run the algorithms:
    - Nonogram: 
        - DFS: `python nonograms_dfs.py`
        - Heuristic: `python nonograms_heuristic.py`

    - Jig-saw sudoku:
        - `python jigsaw_sudoku.py`
